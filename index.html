Commands

General Commands:
!help               Show this help text.
!add                Show link to add bot to your server.
!discord            Show link to join Carl Discord server.
!website            Show link Carl's very own website.
!info               Show uptime and technical information.

Basic Commands:
!gif [term]         Giphy API search - random GIF.
!rif [term]         Riffsy API search - random GIF.
!urban [term]       Urban dictionary search.
!google [term]      Google CSE API search.
!youtube [term]     Google YouTube API search.
!weather [term]     OpenWeatherMap API search.
!stock [stock]      Yahoo stock API search.
!ask [term]         Wolfram Alpha API search (Beta).
!twitch [user]      Twitch user API search (Beta).

Developer Commands:
!geo [host]         Free Geo IP API search.
!host [host]        Perform gethostbyaddr or gethostbyname.
!whois [host]       Perform a pythonwhois.
